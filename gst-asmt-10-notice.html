How to Respond to a GST ASMT-10 Notice in India

A practical, numbers-first guide by CA Ryan Vaz, FCA — using Python reconciliation to close notices with zero penalty.

What is ASMT-10?

ASMT-10 is a GST scrutiny notice issued under Section 61 of the CGST Act when the tax officer finds discrepancies in your return—typically ITC mismatches between GSTR-2A/2B and your books. You must reply via Form GST ASMT-11 within 30 days.

Step-by-Step Response

1. Do not ignore — note the 30-day window

Missing the deadline escalates to DRC-07 (demand order).

2. Run GSTR-2A/2B reconciliation first

Every ITC line must be matched before drafting. Python automation does this in minutes.

3. Draft ASMT-11 with data mapping

Attach a line-by-line reconciliation, not a letter.

4. If missed: respond to DRC-07

Show cause with the same reconciliation evidence.

Case Study: Mumbai Trading Co.

ASMT-10 received for ₹14L ITC mismatch. Python GSTR-2A/2B engine identified 212 mismatched invoices. Reply filed with mapping. Closed in 21 days, zero penalty.
